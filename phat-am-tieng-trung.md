# Thanh điệu
- Thanh 1 : 5 => 5 :
- Thanh 2 : 3 => 5 : nói nhanh như dấu sắc
- Thanh 3 : 2 => => 4 : hơi giống dấu hỏi, chỉ hất lên khi ko có từ nào đi kèm.
- Khi đứng trước từ có thanh 1, 2, 3, nhẹ thì phát âm còn nửa thanh 3 như dấu hỏi.
- Thanh 4 : 5 => 1 : 
- Thanh nhẹ : đứng trước là 1,2,4 thì sẽ ở cao độ 2, còn thanh 3 thì sẽ tầm cao độ 4.

## Biến âm thanh 3
- 2 thanh 3 đi liền thì thanh 3 đầu sẽ thành thanh 2 : 3 3 => 2 3
- 3 thanh 3 đi liền thì thành 2 2 3 hoặc 3 2 3.
- 4 thanh 3 đi liền : 2 3 2 3
